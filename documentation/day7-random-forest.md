# Day 7 Random Forest Regressor

## Objective

Develop and evaluate a Random Forest Regressor model for insurance charge prediction and compare its performance with previously developed machine learning models.

---

## Model Used

**Random Forest Regressor**

Random Forest is an ensemble learning algorithm that combines multiple decision trees to improve prediction accuracy and reduce overfitting.

---

## Features Used

### Input Features

* age
* sex
* bmi
* children
* smoker
* region

### Target Variable

* charges

---

## Training Process

* Loaded the processed insurance dataset.
* Selected relevant input features and target variable.
* Split the dataset into training and testing sets using an 80:20 ratio.
* Trained a Random Forest Regressor model with 100 estimators.
* Generated predictions on unseen test data.
* Evaluated model performance using standard regression metrics.
* Performed feature importance analysis.
* Compared model performance against previously developed models.

---

## Evaluation Metrics

### Mean Absolute Error (MAE)

**2555.93**

The model's predictions differ from actual insurance charges by approximately 2556 units on average.

### Mean Squared Error (MSE)

**21,420,846.46**

The model achieved a substantially lower prediction error than Linear Regression, although slightly higher than the Decision Tree model.

### R² Score

**0.8834 (88.34%)**

The model explains approximately 88.34% of the variation in insurance charges, demonstrating strong predictive performance.

## Model Comparison

| Metric   | Linear Regression | Decision Tree | Random Forest |
| -------- | ----------------- | ------------- | ------------- |
| MAE      | 4182.35           | 2593.36       | 2555.93       |
| MSE      | 35,493,102.61     | 19,778,189.65 | 21,420,846.46 |
| R² Score | 80.68%            | **89.24%**    | 88.34%        |

### Comparison Analysis

* Random Forest significantly outperformed the Linear Regression baseline model.
* Random Forest achieved the lowest Mean Absolute Error among all models.
* Decision Tree achieved the highest R² score and lowest Mean Squared Error.
* Both tree-based models substantially improved prediction performance compared to Linear Regression.
* Ensemble learning proved highly effective for insurance charge prediction.

---

## Key Findings

* Random Forest achieved an R² Score of 88.34%.
* Average prediction error was reduced to 2555.93.
* Smoking status was identified as the dominant predictive feature.
* BMI and age were the next most influential variables.
* Tree-based models consistently outperformed Linear Regression.
* Decision Tree remains the best-performing model developed so far.

---

## Conclusion

A Random Forest Regressor model was successfully developed and evaluated for insurance charge prediction. The model achieved strong predictive performance with an R² score of 88.34% and the lowest Mean Absolute Error among all tested models. Feature importance analysis confirmed that smoking status, BMI, and age are the primary drivers of insurance costs. Although Random Forest delivered excellent results, the Decision Tree Regressor remains the best-performing model overall based on R² score and Mean Squared Error. These findings provide a strong foundation for model optimization and final system development in subsequent phases of the project.
