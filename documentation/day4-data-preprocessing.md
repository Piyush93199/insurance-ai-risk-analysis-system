# Day 4 Data Preprocessing

## Objective

Prepare the insurance dataset for machine learning model development.

## Activities Performed

* Verified data quality and completeness.
* Checked for missing values.
* Examined duplicate records.
* Encoded categorical variables.
* Performed outlier detection.
* Selected input and target variables.
* Created training and testing datasets.

## Data Cleaning Results

### Missing Values

No missing values were detected.

### Duplicate Records

Duplicate records were identified and removed where necessary.

### Categorical Encoding

The following features were converted into numerical format:

* sex
* smoker
* region

## Feature Selection

### Input Features

* age
* sex
* bmi
* children
* smoker
* region

### Target Variable

* charges

## Dataset Split

* Training Data: 80%
* Testing Data: 20%

## Outcome

The dataset was successfully prepared for machine learning model development and evaluation.
