# Insurance AI Risk Analysis System

## Overview

Insurance AI Risk Analysis System is an AI-powered decision support platform designed to analyze customer information, assess risk levels, and recommend suitable insurance products using machine learning techniques.

The project simulates a real-world insurance analytics workflow by combining data preprocessing, exploratory data analysis, predictive modeling, recommendation systems, and interactive visualization.

This project is being developed as part of a 45-day AI/ML internship focused on applying machine learning concepts to business problems in the insurance domain.

---

## Objectives

* Analyze customer demographic and financial data.
* Predict customer risk categories using machine learning models.
* Recommend suitable insurance products based on customer profiles.
* Generate business insights through data visualization and analytics.
* Build an interactive dashboard for decision support.
